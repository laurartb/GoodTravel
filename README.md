# GoodTravel

Proyecto Web: Londres

Este proyecto es una página web informativa sobre la ciudad de Londres, pensada como un sitio turístico dirigido a un público joven que quiere conocer la ciudad y perder el miedo a visitarla.

La web incluye una portada visual con un carrusel de imágenes, una introducción general sobre Londres, una sección de lugares destacados, un gráfico sobre el clima, un mapa interactivo y un footer con información básica.

El proyecto está desarrollado con HTML, CSS y JavaScript, utilizando jQuery para la interacción y Chart.js para el gráfico. Las tipografías se han añadido mediante Google Fonts y las imágenes provienen de Unsplash y de una galería personal.
